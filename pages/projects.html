Chrome Couch

UofTHacks 2015

A chrome extension that allows you to use an X-Pad game controller to browse the internet. It was my first experience with Javascript and asynchronous programming. I worked with the Chrome.* API to implement tab management features and worked with it's message passing system.

aSymple Platformer

Fall GI Jam 2014

A 2-player platformer game made using the Unity Game Engine and C#. We built a platformer where "one player screws over other as hard as possible." While one player is trying to reach the end of the level, the other is able to place obstacles to try and stop them. I worked on scripting obstacle and enemy behaviour, implementing balancing features such as drop zones and timers, and did the art.

MyoPad

Waterloo EngHack 2014

An application that transforms coordinates from a Myo Armband into a digital drawing. This was made using C++/OpenGL, and was an opportunity for me to learn about the graphics pipeline. I worked on using OpenGL to draw any processed coordinates/lines onto an actual GUI.

Blind

Winter GI Jam 2015

An RPG made using the Unity Game Engine and C#. The idea behind this RPG was to never give the player any solid information, only vague messages. A player won't know exactly how much health they have, or how much damage they can do. This game was meant as an experiment in saving game states and moving between scenes in Unity, and remains incomplete. I worked on the battle system, as well as the art.
